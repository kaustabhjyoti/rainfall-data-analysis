# Rainfall Data Analysis and Prediction

## About the Project

This was my first Data Science project. The goal of this project was to analyze rainfall data and understand how machine learning can be used to predict rainfall values.

## What I Did :-

* Loaded rainfall data using Pandas
* Explored the dataset
* Visualized the data using graphs
* Split the data into training and testing sets
* Trained a Linear Regression model
* Predicted rainfall values
* Evaluated the model using the R² score

## Technologies Used :-

* Python
* Pandas
* Matplotlib
* Scikit-Learn

## What I Learned :-

Through this project, I learned about:

* Data analysis
* Train-Test Split
* Linear Regression
* Model evaluation
* Basic machine learning workflow

## Future Improvements :-

In the future, I would like to use larger rainfall datasets and compare different machine learning models.
